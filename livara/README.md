# Livara

نسخة React/Vite جاهزة للنشر على Vercel.

## تشغيل محلي

```bash
npm install
npm run dev
```

## النشر على Vercel

```bash
npm install -g vercel
vercel login
vercel --prod
```

أو ارفع المجلد إلى GitHub ثم اربطه من لوحة Vercel.

## ما تم تفعيله بدون تغيير التصميم العام

- حفظ تخطي شاشة البداية في localStorage.
- نشر إعلان جديد من نموذج إضافة الإعلان.
- حفظ الإعلانات محلياً.
- صفحة المفضلة فعالة.
- صفحة الحساب فعالة.
- البحث يشمل العنوان والولاية والوصف والمميزات.
- رابط واتساب محسّن مع noopener/noreferrer.
- المشروع مجهز بملفات Vite و Vercel.

> ملاحظة: التحقق من الهوية هنا ما زال محاكاة محلية داخل المتصفح، وليس تحققاً حكومياً حقيقياً أو Backend production.
